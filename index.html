1. Introdução

Este Relatório descreve o projeto de controle de um Led com a utilização de um sensor infravermelho e um Arduino, com o objetivo de desenvolver habilidades em eletrônica e programação.

2. Objetivos

O principal objetivo é fazer o controle do led RGB com a utilização de um sensor infravermelho em um microcontrolador Arduino.

3. Materiais utilizados

  • Arduino
  • LED normal ou LED RGB
  • 1 Sensor infravermelho reflexivo
  • 1 Resistor
  • Protoboard
  • Fios de conexão

4. Código Fonte

            const int PinLed = 9;
const int PinSensor = 8;
int LeituraSensor;

void setup() {
  Serial.begin(9600);
  pinMode(PinSensor, INPUT);
  pinMode(PinLed, OUTPUT);
}

void loop() {
  LeituraSensor = digitalRead(PinSensor);
  Serial.print("Leitura do sensor: ");
  Serial.println(LeituraSensor);
  
  if (LeituraSensor == LOW) {
    digitalWrite(PinLed, HIGH);
  }
  else {
    digitalWrite(PinLed, LOW);
  }

  delay(100);
}

5. Procedimentos

O sensor infravermelho e o LED foram conectados aos pinos digitais do Arduino conforme o diagrama, e o código foi carregado na placa.

6. Resultados

O LED se acendia e apagava conforme um objeto não reflexivo/reflexivo passava em frente ao sensor.

7. Discussão

O funcionamento do circuito aconteceu de forma satisfatória e conforme o esperado.

8. Conclusão

Após muitas tentativas e erros o projeto foi bem sucedido, permitindo a pratica dos conceitos de eletrônica e programação.

Perguntas

  1. Qual o valor da corrente elétrica de um LED normal e um LED RGB? Um LED normal geralmente opera com uma corrente de cerca de 20 mA, enquanto um LED RGB pode consumir até 60 mA quando os três LEDs estão acesos simultaneamente.
  2. Qual o valor da corrente elétrica de um sensor infravermelho reflexivo? Um sensor infravermelho reflexivo geralmente consome entre 1 mA e 20 mA, dependendo do modelo específico.
